Steps to create an End to End ML Project:

1. Data Collection
  > Sources for data collections:
    - Web Scrapping
    - Surveys
    - Database or Logs
    - APIs
    - Public Datasets

2. Data Inscpection
    - Summary of dataset
    - Checking for Null Values, Data Type of Features, 

3. Data Cleaning
    - Handling Missing Data
    - Removing redundant and duplicate data
    - Handling untidy dataset etc
 
4. Exploratory Data Analysis
    - Finding Correlations
    - Univariate Analysis
    - Bivariate Analysis
    - Multivariate Analysis
    - Checking for multicolinearity
    - Understanding about each and every features/columns/variables and putting our valuable observations

5. Feature Engineering
    - Handling Categorical features
    - Feature Derivation
    - Selecting top/important features

6. Model Selection & Training
    - Scaling Data
    - Splitting Data
    - Selecting Modal
    - Training the Modal
    - Modal Evaluation and Hyperparameter Tuning

------------------------------------------------------------

7. Creating REST API or UI

8. Pipelining & Deployment
